# SS.INN OYO & Hotels — Website

React + Vite + Tailwind landing page for SS.INN OYO & Hotels, Gopanpally, Hyderabad.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build (outputs to `dist/`)
- `npm run preview` — preview the built site
- `npm run deploy` — publish `dist/` to GitHub Pages

## Deploy

Hosted via GitHub Pages at https://salesforcecourse9-bot.github.io/.
Run `npm run deploy` to build and publish to the `gh-pages` branch.
